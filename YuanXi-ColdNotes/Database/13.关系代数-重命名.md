## $\rho_{S(A_1,...,A_n)}(R)$
- 重命名 (Rename)
- 用于将关系$R$中的属性重命名为新属性名$S(A_1, ..., A_n)$，以便在后续操作中使用。

$R$

|     A     |     B     |     C     |
| :-------: | :-------: | :-------: |
|   $a_1$   |   $b_1$   |   $c_1$   |
| **$a_1$** | **$b_2$** | **$c_2$** |
|  *$a_2$*  |  *$b_2$*  |  *$c_1$*  |

$\rho_{S(A_1, B_1, C_1)}(R)$

|   $A_1$   |   $B_1$   |   $C_1$   |
| :-------: | :-------: | :-------: |
|   $a_1$   |   $b_1$   |   $c_1$   |
| **$a_1$** | **$b_2$** | **$c_2$** |
|  *$a_2$*  |  *$b_2$*  |  *$c_1$*  |
