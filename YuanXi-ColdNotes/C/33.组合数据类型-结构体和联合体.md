## 结构体（Structure）
把不同的类型组合到一起,形成一个新的类型
### 格式
```c
struct Student{
	int id;
	char[50] name;
	float height;
}
```
### 使用
```c
struct Student{
	int id;
	char[50] name;
	float height;
}

struct Student s1 = {123,"lihua",1.83};
printf("id:%d, name:%s, height:%f m", s1.id, s1.name, s1.height);
```
- 如果在初始化结构体时缺值,缺少的值会被默认初始化为垃圾值
